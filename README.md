# Eswatini MTN - Digital SIM Services Demo v5.0

This is a demonstration prototype of Eswatini MTN's SIM Registration and SIM Swap services.

## Features
- Customer login/signup
- SIM Registration
- SIM Swap
- Admin login/dashboard
- Approve/Reject requests
- Live pie chart analytics
- Fraud risk score and cost savings
- Export report to PDF

## How to Host on GitHub Pages
1. Create a GitHub repository named `eswatini-mobile-demo`.
2. Upload all files from this folder (`index.html` and `README.md`).
3. Go to **Settings > Pages** in your repository.
4. Under **Source**, select `main` branch and `/ (root)` folder.
5. Click **Save**.
6. Your demo will be live at:  
   `https://<your-username>.github.io/eswatini-mobile-demo/`

## Usage
- Customer portal:
  - Sign up with a username/password
  - Log in and try SIM Registration and Swap
- Admin portal:
  - Login with:
    - Username: `admin`
    - Password: `Admin@123`
  - Approve/Reject requests
  - View pie chart analytics and export PDF report
